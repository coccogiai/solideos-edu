# System Monitoring Edu Project

Real-time system resource monitoring software developed for educational purposes.

## Features
- **Real-time Monitoring**: Tracks CPU, GPU, Memory, Disk, and Network usage.
- **Visual Dashboard**: Intuitive UI with charts and tables.
- **Reporting**: Generate PDF reports of 5-minute tracking sessions.

## Installation & Usage

1. Install dependencies:
   ```bash
   pip install -r system_monitor/requirements.txt
   ```

2. Run the application:
   ```bash
   python system_monitor/app.py
   ```
